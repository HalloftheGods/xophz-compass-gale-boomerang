# Xophz Magic Boomerang

> **Category:** True North · **Version:** 0.0.1

Catch wind of where most of your returning traffic is visiting from.

## Description

**Gale Boomerang** (Magic Boomerang) is a return-visitor analytics plugin for COMPASS. It identifies and tracks returning users, mapping their referral sources and visit patterns to reveal which channels consistently bring people back.

### Core Capabilities

- **Return Traffic Analysis** – Identify which sources drive the most repeat visitors.
- **Referral Mapping** – Track referral chains across multiple visits.
- **Visit Frequency** – Measure how often users return and what triggers re-engagement.
- **Source Comparison** – Side-by-side analytics of referral sources by loyalty metrics.

## Requirements

- **Xophz COMPASS** parent plugin (active)
- WordPress 5.8+, PHP 7.4+

## Installation

1. Ensure **Xophz COMPASS** is installed and active.
2. Upload `xophz-compass-gale-boomerang` to `/wp-content/plugins/`.
3. Activate through the Plugins menu.
4. Access via the My Compass dashboard → **Gale Boomerang**.

## Frontend Routes

| Route | View | Description |
|---|---|---|
| `/gale-boomerang` | Dashboard | Return traffic analysis and referral source mapping |
